This page is a local Genipedia test article used to match the Genipedia person-page layout.

Shaun Roselt is the sample biography subject used to test the full Genipedia-style header, footer, and article layout in Genipedia. This page exists as a local reference surface so the person-page shell can be tuned against a real article structure instead of an empty document.

In the current test layout, the Shaun Roselt page is used to verify the desktop header grid, search form placement, appearance controls, page-title toolbar, article tabs, and footer spacing. It is intentionally simple, but it behaves like a normal article page so component work can be validated in context.

Testing Role

This article is the primary offline test surface for the full-header and full-footer components. It makes it easier to compare the local Genipedia shell with the live Genipedia Nelson Mandela page while keeping the project fully local and dependency-free.

Layout Notes

The page is designed to expose header behavior clearly: the title row sits directly under the top header, the action tabs remain visible, and the lead text gives the article body enough structure to make spacing and borders easy to compare during UI work.
