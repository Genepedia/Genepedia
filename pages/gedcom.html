Overview

A GEDCOM file describes people, families, events, and relationships in a format most genealogy applications understand. On {{APP_NAME}}, each profile's family tree can be exported as GEDCOM or built up by importing data from another program.

Use Import to bring tree data into {{APP_NAME}}, Export to download a profile's tree for backups, or Tree Viewer to inspect a GEDCOM file before deciding what to do with it.
