← GEDCOM Import and Export

A GEDCOM export downloads a profile's family tree as a standard file you can open in other genealogy software. The export is centred on one person—the profile you open.

How to export a GEDCOM file

From any profile with a family tree:

  1. Open the person's profile page.
  2. Select the Tree tab.
  3. Open the toolbar Download menu.
  4. Choose GEDCOM to save the family tree file.

If you are signed in and the profile is yours, you can also reach it from the user menu. To export a tree centred on someone else, use Search or browse profiles to open their page first.

What is included

The download contains the profile's family tree data—people, relationships, and key life events. You can open it in desktop genealogy software, upload it to another website that accepts GEDCOM, or keep it as a backup.

Biography text, photos, and sources on the profile and page tabs are not included; only the structured tree data is exported.

If export is unavailable

If the Tree tab says no family tree is available yet, open the profile editor, fill in the identity details, and save. That creates the tree so you can add relatives and export later.
