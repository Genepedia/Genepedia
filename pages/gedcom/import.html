A GEDCOM import brings people, families, and events from another genealogy program into {{APP_NAME}}. Imports are always focused on a particular person—the profile who anchors the branch you are building.

How to import a GEDCOM file

Choose the path that matches your situation:

Start a new branch

If your GEDCOM file contains people who are not yet on {{APP_NAME}}, create a profile for one of them and grow the tree from there.

  1. Open New Tree from the site menu and create a profile for the person who should anchor the import.
  2. Fill in the identity details and save the profile.
  3. Add relatives on the profile's Tree tab as you build out the branch.

Extend an existing profile

If your GEDCOM file includes people who already have profiles on {{APP_NAME}}, open that person's profile first. Use the Tree tab to review what is already recorded, then add missing relatives and events through the profile editor rather than duplicating existing profiles.

Upload a GEDCOM file

Direct GEDCOM file upload—with matching to existing profiles and guided merge—is coming in a future release. For now, start with New Tree or extend an existing profile as described above. If you need help with a large import, contact {{APP_NAME}}.

Tip: Because a large import can create duplicate profiles that already exist on {{APP_NAME}}, work in small sections—ancestors and descendants around one focus person at a time—and resolve matches before continuing on that branch.
